# Contributing

Thank you for improving VC Sourcing Funnel.

## Ground rules

- Respect each source site's terms of use, robots directives, rate limits, and applicable privacy laws.
- Do not submit features that collect private contact details or send messages without a human approval step.
- Keep source adapters narrowly scoped and add fixture-based tests for parsing changes.

## Pull requests

1. Create a focused branch.
2. Run `npm test`.
3. Explain the source, extraction assumptions, and how the change was tested.
